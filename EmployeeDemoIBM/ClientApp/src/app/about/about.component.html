Project highlights and demo

  • This project is build using WebAPI Core and Angular8
  • Sql Server local database provided with visual studio is used as database

.Net WebAPI Core

  • MVC architectural pattern is followed.
  • Repository pattern is followed along with .NetCore in-built dependency injection, so that controller talks with repository and not with actual database context classes.
  • Followed SOLID principle and loosely coupled objects across application.
  • Followed REST principles while designing API
  • Implemented Logger using Micosoft extensions logging
  • Unit test cases is build using .NetCore xUnit (Framework is built and 4 sample test cases has been written for demo)
  • Thought of adding JWT Authentication, but due to time constraint, couldn't complete that part.

Angular8

  • Followed component oriented architecture
  • Angular Material + Bootstrap is used
  • Observables
  • Created reusable components
  • Basic front end add on functionalities like pagination, popup models has been used
